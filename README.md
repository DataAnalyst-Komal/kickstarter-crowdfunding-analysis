# Kickstarter Crowdfunding Analysis Dashboard

## Overview

This project analyzes Kickstarter crowdfunding campaigns to evaluate project performance, funding trends, campaign success rates, and geographic contributions. The dashboard provides actionable insights to help understand the factors influencing successful crowdfunding campaigns.

## Objective

To analyze crowdfunding performance across project categories, funding goals, campaign durations, and countries, identify key success factors, and uncover trends that support data-driven business decisions.

## Tech Stack

- **Power BI** — Dashboard design, data visualization & interactive reporting
- **Power Query** — Data cleaning & transformation
- **DAX** — Calculated columns & measures

## Dataset

- **Source:** Kaggle (Kickstarter Projects Dataset)
- **Period:** 2009 – 2018
- **Fields include:** Project name, category, country, launch date, deadline, funding goal, pledged amount, backers, campaign duration and project outcome

## KPI Summary

| Metric | Value |
|--------|-------|
| Total Projects | 363K |
| Success Rate | 38.50% |
| Total Amount Raised | $3.85B |
| Total Backers | 44M |
| Avg Funding per Successful Project | $27.54K |

## Key Insights

- **Only 38.5% of projects achieved their funding goals**, highlighting the competitive nature of crowdfunding.

- **Projects with funding goals below $5K recorded the highest success rate (48.84%)**, while success declined significantly as funding goals increased.

- **Product Design generated the highest total funding**, followed by Tabletop Games, making them the strongest performing categories.

- **The United States dominated Kickstarter funding**, with most crowdfunding activity concentrated in developed economies.

- **Successful campaigns averaged only 32 days**, suggesting shorter campaign durations are associated with better outcomes.

## Dashboard Features

- Overview of crowdfunding performance and funding trends
- Campaign success analysis by funding goal, category and campaign duration
- Top-performing categories and countries by funding and success rate
- Geographic funding distribution using an interactive map
- Interactive filters for Year, Quarter and Month
- Business insight panels with executive summaries across all dashboard pages
- Four-page dashboard navigation for comprehensive analysis

## Business Impact

- Identifies factors associated with higher crowdfunding success
- Supports creators in setting realistic funding goals and campaign durations
- Highlights high-performing categories for strategic investment
- Reveals geographic funding patterns and potential growth opportunities
- Enables stakeholders to monitor crowdfunding performance through interactive reporting
